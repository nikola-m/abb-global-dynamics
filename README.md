# abb-global-dynamics
The research-grade framework for the analysis of global dynamics and stability (e.g. coexisting attractors, Hopf bifurcation boundary) for a nonlinear rotor system with an Automatic Ball Balancer (ABB).

The code is used to obtain the numerical results in the paper:
Mirkov, N.; Pezo, M.; Jovanović, R.; Balać, M.; Peković, O. Global Dynamics and Stability of Automatic Ball Balancers Under Anisotropy and Non-Ideal Excitation. Modelling 2026, 7, 135. https://doi.org/10.3390/modelling7040135

If you find the code useful, please cite our paper :)
