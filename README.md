# abb-global-dynamics
The research-grade framework for the analysis of global dynamics and stability (e.g. coexisting attractors, Hopf bifurcation boundary) for a nonlinear rotor system with an Automatic Ball Balancer (ABB).

The code is used to obtain the numerical results in the paper:
N. Mirkov, M. Pezo, R. Jovanovic, M. Balac, O. Pekovic. `Global dynamics and stability of automatic ball balancers under
anisotropy and non-ideal excitation` (Submitted to MDPI Modelling).
